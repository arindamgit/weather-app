# Weather App
In this projecct I test ESP8266 Module step by step and attempt to build a weather display device
The goal is to pull weather information form the internet and display that on a Nextion display
General Notes
## ESP8266 Connection Notes
* GPIO-0 must be held low during flashing
* CH_PD must be held high during normal operation
* Sometimes the RESET pin is required to be held low during flashing
